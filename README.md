# AI Weather + Trip Planner

An intelligent AI assistant that combines weather forecasting and web search to answer travel-related questions.

## Features

- Weather information using Open-Meteo API
- Web search using DuckDuckGo
- Intelligent routing using Groq Llama 3.3
- Gradio Chat Interface
- AI-generated travel suggestions

## Technologies

- Python
- Groq API
- DuckDuckGo Search
- Open-Meteo API
- Gradio

## Installation

```bash
pip install groq duckduckgo-search gradio requests
```

## Run

```python
python app.py
```

Example:

```
Should I carry an umbrella for Goa and suggest places to visit?
```
